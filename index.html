What I’ve Built

Latest Projects

Full-Stack Task Management Web Application

Designing and implementing a full-stack task management web application using React, Node.js, JavaScript, HTML, and CSS, with a clean, responsive UI and intuitive UX. The system integrates both NoSQL (MongoDB) and SQL (PostgreSQL) for efficient data storage and querying, exposes RESTful APIs for task and user management, and incorporates AWS cloud services for backend integrations (non-deployment). The application supports secure user authentication, task creation and organization, due-date tracking, and scalable handling of 100+ tasks per user, demonstrating strong full-stack engineering, database design, and real-world application architecture.

Full-Stack Expense Tracker Web Application

A full-stack web app built with Python and Flask that allows users to securely log expenses, store data in a SQL database, and view weekly spending totals and category breakdowns through interactive visualizations. The project features user authentication, real-time data aggregation, a responsive UI, and is configured for cloud-ready deployment, demonstrating end-to-end web development and data analysis skills.
